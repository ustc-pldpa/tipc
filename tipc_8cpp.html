tipc driver.

This function is the entry point for tipc. It handles command line parsing using LLVM CommandLine support. It runs the phases of the compiler in sequence. If an error is detected, via an exception, it reports the error and exits.
If there is no error, then the LLVM bitcode is emitted to a file whose name is the providvvved source file suffixed by ".bc".
